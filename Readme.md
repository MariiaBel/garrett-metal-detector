### Памятка

#### 1. Первый запуск

Установите используемые пакеты  командой 
```
npm install
```
#### 2. Сборка проекта

```
npm run dev //собирает проект в папку `build/` и включает синхронизацию браузера с папкой `build/`
```

```
npm run build //собирает проект в папку `build/
```
